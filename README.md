# decodelabs_task_03
Project 03: Random Password Generator

Description

The Random Password Generator is a Python-based application that generates secure and unique passwords. Users can enter a keyword, birth date, and desired password length to create a customized password with random letters, numbers, and special characters.

Technologies Used

- Python 3

Features

- User-defined keyword
- Birth date integration
- Custom password length
- Random letters, numbers, and special characters
- Secure password generation
